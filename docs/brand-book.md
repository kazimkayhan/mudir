# Mudir Brand Book

## Brand foundation

**Name:** Mudir (مدیر) — Dari/Persian for *manager/director*.

**Positioning (Dari):**
> مدیر — نرم‌افزار ساده و قابل اعتماد برای مدیریت فروش، موجودی، خریداری و گزارش‌های روزانه فروشگاه‌ها در افغانستان.

**Positioning (English):**
> Mudir — trusted offline retail management for Afghan stores: in-store sales, online orders, stock, purchases, and daily reports.

**Tagline:**
- Dari: **مدیریت فروشگاه، آسان و آفلاین**
- English: **Store management, simple and offline**

**Terminology rule:** Always use **فروشگاه** — never **مغازه**.

## Receipt standard

- Store name (نام فروشگاه), address, phone
- Receipt #, localized date/time
- Line items with currency
- Footer: «سپاس از خرید شما» / «Thank you»
